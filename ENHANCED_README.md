# Multi-Chain Stablecoin Super App

A full-featured stablecoin super application with unified balance view, seamless cross-chain transfers, yield generation, payments, and on/off-ramp integration.

## Features
- Unified multi-chain stablecoin dashboard
- CCTP-powered cross-chain transfers
- Yield-bearing stablecoin options
- Built-in payments and invoicing
- On/off-ramp integrations

## Architecture
- Core wallet and balance contracts
- Cross-chain transfer module
- Yield strategy integration
- Payment and invoicing layer
- Frontend and mobile support

## Getting Started
Follow the repository structure and deployment guides for supported chains.

## Vision
Become the go-to interface for stablecoin usage across chains.
